## General info

This project is a simple page which allows to see and rate cute animals.

## Technologies

Project is created with:

- React 18.2.0,
- React-dom 18.2.0,
- React-redux 8.0.5,
- Redux 4.2.1,

## Setup

To run this project, install it locally using npm:

```
$ cd ../cute-animals
$ npm install
$ npm start
```
